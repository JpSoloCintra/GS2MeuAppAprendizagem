O Futuro do Trabalho – Aplicativo Android com Kotlin, Compose e Oracle

Este projeto foi desenvolvido para a Global Solution 2 – FIAP, com o objetivo de criar uma solução tecnológica que contribua para a adaptação ao futuro do trabalho.

A aplicação permite:

Listar candidatos

Consultar pontuação calculada por IA/algoritmos (backend)

Acessar métricas de aprendizagem geradas automaticamente no banco Oracle usando PL/SQL

Tecnologias Utilizadas
1. Kotlin (Android)

Linguagem moderna e segura

Menos código que Java

Coroutines para chamadas assíncronas sem travar a interface

Jetpack Compose para criar UI moderna e declarativa

2. Retrofit + OkHttp

Permite comunicação com backend REST:

Simples

Confiável

Conversão automática de JSON

3. Jetpack Compose

Criação das telas de forma:

Rápida

Intuitiva

Facilmente escalável

4. Banco Oracle + PL/SQL

Utilizado para automação de processos:

Procedures para registrar atividades

Funções para calcular score

Jobs para atualizar métricas diariamente

Controle de desempenho e uso
